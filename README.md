# PF Mitra Static Website

A high-conversion, premium, mobile-first static website for PF Mitra (PF Withdrawal Assistance).

## Directory Structure
- `index.html`: The main landing page.
- `guide.html`: Step-by-step PF withdrawal guide (fully responsive interactive accordion).
- `privacy.html`: Privacy policy page.
- `terms.html`: Terms of Service page.
- `_redirects`: Netlify routing configuration for clean URLs (e.g. `/guide` instead of `/guide.html`).
- `.gitignore`: Configured to ignore system temporary files.

## Running Locally
To test the website locally:
1. Open any page (e.g. `index.html`) in a browser by double-clicking the file.
2. Alternatively, run a local development server using Node.js:
   ```bash
   npx serve .
   ```
   Or using python:
   ```bash
   python -m http.server 8000
   ```

## Deploying to Netlify (Free Hosting)

### Step 1: Initialize Git Local Repository
Open your terminal (PowerShell, Command Prompt, or Git Bash) in this project folder (`c:\Users\Dhruv\OneDrive\Consultancy\Folder`) and run:
```bash
git init
git add .
git commit -m "Initial commit of refined PF Mitra website"
```

### Step 2: Create a GitHub Repository and Push
1. Go to [GitHub](https://github.com) and sign in.
2. Click **New** (or go to [github.com/new](https://github.com/new)) to create a new repository.
3. Name it `pf-mitra` (or any name you prefer). Leave it **Public** (or Private) and do **NOT** check "Add a README", "Add .gitignore", or "Choose a license".
4. Click **Create repository**.
5. Copy the commands under "**…or push an existing repository from the command line**". They will look like this:
   ```bash
   git branch -M main
   git remote add origin https://github.com/YOUR_GITHUB_USERNAME/pf-mitra.git
   git push -u origin main
   ```
   *(Replace `YOUR_GITHUB_USERNAME` with your actual GitHub username).*
6. Run those commands in your terminal to push the code to GitHub.

### Step 3: Deploy to Netlify
1. Go to [Netlify](https://www.netlify.com) and sign up/sign in (using your GitHub account is easiest).
2. Once logged in, go to your dashboard and click **Add new site** > **Import an existing project**.
3. Select **GitHub** as the provider (authorize Netlify if prompted).
4. Select your `pf-mitra` repository.
5. In the Site settings:
   - **Branch to deploy**: `main`
   - **Build command**: *Leave empty* (since it is a static HTML site)
   - **Publish directory**: `.` (or leave it blank)
6. Click **Deploy pf-mitra**.
7. Netlify will deploy the site in a few seconds. It will give you a temporary random domain name (e.g. `https://wonderful-cookies-12345.netlify.app`).

### Step 4: Configure Custom Domain
Once you purchase your domain:
1. In the Netlify dashboard for your site, go to **Site configuration** > **Domain management**.
2. Click **Add custom domain**.
3. Enter your purchased domain (e.g., `pfmitra.in` or `pfmitra.com`).
4. Netlify will show you the DNS records (usually an ALIAS/CNAME or Name Servers) to add to your domain registrar (like GoDaddy, Namecheap, Hostinger, etc.).
5. Once DNS updates (can take from 5 minutes to 24 hours), your site will be live on your custom domain with a free SSL certificate automatically generated by Netlify!
